# biostats_project
Data Analysis with NHAHES blood pressure data
This project takes blood pressure data from the 2017-March 2020 and 2021-2023 cycles in NHANES as well as demographic data from the latter cycle
This project has two goals
1. Determine the difference (if any) in the blood pressure values between the 2 study cycles, seeing how the COVID-19 pandemic may have shaped said difference
2. Determine the demographic factors (age, level of education, gender, weight) that impact the blood pressure values (both systolic and diastolic)
